About CSS

How to change colors

background-color Changes the background color
color Changes the text color

How to change texts

font-size Sets the font size
font-family Sets the font type
font-weight Sets the font weight (boldness)
text-align Aligns the text (left, right, center)

How to work with spacing

margin Sets the space outside an element
padding Sets the space inside an element
border Adds a border around an element

Basic uses of CSS

CSS (Cascading Style Sheets) is used to style and design web pages. While HTML gives a page its structure, CSS makes it look visually appealing. It controls how content is displayed, including colors, spacing, layout, and fonts.

  • Styling Text

    You can change the color, size, font, and alignment of text.

  • Setting Colors and Backgrounds

    CSS allows you to change the background color of elements or the entire page.

  • Adding Spacing

    With margin and padding, you can control the space outside and inside elements.

  • Creating Layouts

    CSS helps you arrange items on the page using layout tools like flexbox and grid.

  • Making Things Responsive

    You can adjust the style of a website depending on the screen size to make it mobile-friendly.

  • Visual Effects

    You can add effects like shadows, rounded corners, and smooth animations to enhance the design.
